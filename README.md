* [GO Utils library](https://github.com/polyglotDataNerd/zib-Go-utils)
* [Terraform](https://learn.hashicorp.com/terraform/getting-started/install.html)
* [AWS Managed Cassandra Service](https://aws.amazon.com/mcs/)
